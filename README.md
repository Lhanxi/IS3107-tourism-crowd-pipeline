## Setup Instructions

### Step 1: Add the GCP Key
1. Unzip the **keys** folder.
2. Place the unzipped **gcp-key.json** file in the following directory:
(`projectroot/`)

### Step 2: Add the Environment File
1. Add a **.env** file to the project root directory (`projectroot/`).
- Ensure that the `.env` file contains the necessary environment variables for the project to run.
